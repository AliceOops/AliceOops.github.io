Java TCP双端通信
发表于 2024-06-23 |     中国|      你好 · 我好
TCP三次握手的过程如下： 1. 客户端发送SYN（SEQ=x）报文给服务器端，进入SYN_SENT状态。 2. 服务器端收到SYN报文，回应一个SYN （SEQ=y）ACK（ACK=x+1）报文，进入SYN_RECV状态。 3. 客户端收到服务器端的SYN报文，回应一个ACK（ACK=y+1）报文，进入Established状态。
